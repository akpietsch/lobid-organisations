JSON-LD

All API calls serve JSON-LD (JSON for Linking Data). You can experiment with that by pasting responses or URLs in the JSON-LD Playground.

Use a JSON-LD processor to convert responses to RDF, e.g. to N-Quads, an N-Triples superset, with jsonld-cli:

jsonld format --nquads http://beta.lobid.org@routes.Application.get("DE-6")

Reconciliation

The lobid-organisations service implements the OpenRefine Reconciliation API for multi query requests.

The service supports details from additional columns during reconciliation, under arbitrary property names.
